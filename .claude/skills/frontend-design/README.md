# Frontend Design Skill

Installed from Anthropic's official `frontend-design` plugin
(https://github.com/anthropics/claude-code/tree/main/plugins/frontend-design).

Loaded by Claude Code whenever frontend / UI work is requested. Guides toward
distinctive typography, color, motion, and layout — and away from generic AI
aesthetics.

## Usage

Just ask for the UI change you want. The skill instructions are picked up
automatically. Example prompts:

- "Make the inbox feel like a high-end magazine."
- "Give this dashboard a brutalist edge."
- "Redesign the compose screen with refined minimalism."
